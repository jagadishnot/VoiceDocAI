# 🤖 VoiceDoc AI

🚀 AI-powered assistant for:
- 📄 Document Question Answering (RAG)
- 📊 Excel Data Analytics
- 🔊 Voice Output (Text-to-Speech)
- ⚡ Real-time Streaming Responses

---

## 🔥 Features

- ✅ Offline AI using Ollama (100% privacy safe)
- ✅ FAISS-based vector search (RAG)
- ✅ Real-time streaming like ChatGPT
- ✅ Voice input & output support
- ✅ Automatic Excel dashboard generation
- ✅ PDF report export

---

## 🧠 Tech Stack

### Backend
- FastAPI
- SentenceTransformers
- FAISS
- Ollama (phi3:mini)
- Piper TTS

### Frontend
- React.js
- Recharts
